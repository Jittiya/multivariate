อธิบายการใช้งาน Github และ Google colab ว่าหน้าที่ของแต่ละอันคืออะไร และอธิบายวิธีเซฟไฟล์จาก Google colab ลงใน Github

Github เป็นเว็บเซิฟเวอร์ที่ให้บริการในการฝากไฟล์ Git (ทั่วโลกนิยมใช้ในการเก็บโปรเจ็ค Open Source). ส่วน Git คือ Version Control ตัวหนึ่ง
ซึ่เป็นระบบที่มีหน้าที่ในการจัดเก็บการเปลี่ยนแปลงของไฟล์ในโปรเจ็คเรา มีการ backup code ให้เรา สามารถที่จะเรียกดูย้อนกลับไปดูเอร์ชั่นต่างๆ ของ
โปรเจ็คที่ใด เวลาใดก็ได้ 
Google colab สามารถใช้ colab สร้าง Note book เขียนภาษา Python ได้ฟรี

วิธีเซฟไฟล์จาก Google colab ลงใน Github
1. เปิด Google colab แล้วกดเปิดไฟล์งาน
2. คลิกคำว่า ไฟล์
3. คลิก บันทึกสำเนาใน Github
4. เลือกที่เก็บ เขียนข้อความคอมมิต คลิก รวมลิงก์ไปยัง Colaboratory
5. คลิกตกลง จากนั้นไฟล์นี้ก็จะไปอยู่ใน Github ของเรา
